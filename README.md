# grss

## Supports

- [ ] JSON Feed Version 1.0
- [ ] JSON Feed Version 1.1
- [ ] Atom 0.3
- [ ] Atom 1.0
- [ ] RSS 0.90
- [ ] RSS 0.91 (netscape)
- [ ] RSS 0.91
- [ ] RSS 0.92
- [ ] RSS 1.0
- [ ] RSS 2.0
- [ ] RSS 2.0.1
- [ ] MRSS 1.5.1

## Features

- [ ] JSON to RSS 2.0
- [ ] JSON to JSON 1.1
- [ ] JSON to Atom 1.0
- [ ] RSS to RSS 2.0
- [ ] RSS to JSON 1.1
- [ ] RSS to Atom 1.0
- [ ] Atom to RSS 2.0
- [ ] Atom to JSON 1.1
- [ ] Atom to Atom 1.0
- [ ] Custom Encoding
